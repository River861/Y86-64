#############################################

River's Y86-64（有GUI版）使用说明书：

#############################################

1 运行功能

a. 运行全部测例：
    方法： 点击Run>>Run all
    效果：1）运行结束后会弹出对话窗口，在其中选择您想查看的运行过程即可
             2）点击Select, 上方显示框显示运行过程；点击主窗口result，查看运行结果
             3）对话窗口不会自动关闭，便于查看其它测例的运行过程和结果

b. 运行单个测例：
    方法：点击Run>>Run one test... 
    效果：1）会弹出输入框，填入您想运行的测例名称（如prog1.yo），再点击Run即可
             2）同样点击主窗口result，查看运行结果

c. 自行输入编码指令：
    方法：在下方编辑框中输入Y86-64指令集中的指令，
             然后点击submit （或者点击Run>>Submit a code...）即可
    输入规范：不允许有空行
                    每一行允许有注释，但不能只有注释
             （如：0x000: 30f20a00000000000000 |   irmovq $10,%rdx）
-----------------------------------------------------------------------------------------------------
2 调试功能

a. 调试测例：
    方法：点击File>>Open a file
              在test cases中选择你想调试的测例，打开
              删除空行（以及只有注释的行）
              在代码中任意两行间可以加入断点（以一行一个break表示）
              点击submit
    效果：上方显示框中显示运行过程，其中在断点处会打出当前状态，并以It's a break point here标识

b.调试自定义的编码指令：
    方法：按输入规范输入指令后，同样在代码中任意两行间可以加入断点（以一行一个break表示），然后点击submit
    效果：结果同上
              编辑完成后可以点击File>>Save the file 保存自己编辑的指令码，以供下次使用
-----------------------------------------------------------------------------------------------------
3 设置功能
    方法：点击Setting>>Setting...
              分别拖动滑条或直接键入数字 设置最大内存和运行速度
    效果：最大内存和运行速度会分别影响代码运行的结果和运行的时间
              显示框每次显示运行过程和结果时 会首先显示当前设置的内存和速度
-----------------------------------------------------------------------------------------------------
4 登陆/注销功能
    方法：点击Log_in>>Log in
              输入用户名和密码，再点击log in即可登陆
              点击Log_in>>Log out即可注销
    效果：登陆后能看见原本看不见的东西
-----------------------------------------------------------------------------------------------------
5 帮助功能
    方法1：点击Help>>Ask for help
    效果：在上方的显示框会显示本README文本，便于您快速查看使用说明

    方法2：点击Help>>Get a Qt_code，用手机扫描弹出的二维码
    效果：扫码后就可以用手机阅读本程序的操作说明啦，这样就能边看说明，便运行程序了
-----------------------------------------------------------------------------------------------------
6 其他
    1）clear按钮
    效果：可以清空上下两个编辑框，并使程序进入初始状态
              当出现由于不当操作造成的界面混乱时，此按钮可以快速复原
    2）工具栏
    说明：本程序具有强大的工具栏功能，点击工具栏上的按键，就能实现与图标对应的功能
    效果：在熟悉了本程序的功能后，按工具栏能方便不少
    3）快捷键
    说明：一些使用频繁的功能设置了快捷键，具体在菜单栏上有明显提示
    效果：使用频繁的功能运用快捷键能大幅提高工作效率
    4）进度条
    效果：每当运行程序或指令时，进度条都会实时更新运行进度
              可以缓解您等待程序运行时的无聊和郁闷
    5）时间
    效果：在主窗口左下方实时更新当前的北京时间
               方便您快速了解时间（再也不用担心不知道今天几号了）
    6）状态条
    效果：将鼠标移动至任意按键上时，屏幕最下方会显示简要说明（比本说明更方便直接）
    7）了解作者
    方法：点击>>Help>>About the author...
    效果：能使您快速了解到作者的基本信息，并收获作者的欢迎辞
-----------------------------------------------------------------------------------------------------
